# market.utils

market パッケージの共通ユーティリティモジュール。

## 概要

`market` パッケージ内で横断的に使用する汎用ヘルパー関数・クラスを集約するサブパッケージです。

**現在の状態**: 準備済み（`__init__.py` のみ、エクスポートなし）

現時点ではファイルが `__init__.py` のみで、`__all__` は空リストです。
将来的にここにユーティリティ関数を追加していく想定のプレースホルダーです。
