## 3. Dogma v1.0 変更の影響分析

### 3.1 主要変更点と影響

| v1.0変更 | 影響を受けた銘柄 | 影響内容 |
|---------|----------------|---------|
| **「定量はWANT、説明力がMUST」** (Rule 4) | ATCOA, AME, POOL | 定量データが少ない非US/ニッチ銘柄でも、説明力が十分なら50%維持。ATCOA#1は定量不足だが説明力で70% |
| **パターンC拡張（因果関係の混同・拡大解釈）** | NFLX, MSFT, LRCX | NFLX#3(専業モデル), #5(パスワード取締り)でPattern F検出。MSFT#3(Ecosystem)でCA 50%→30%降格 |
| **パターンD再定義（説明力不足）** | POOL#6, MCO#6 | POOL#6 E-commerce barriers 50%→30%（「業界共通+補完要素」）。MCO#6 Brand 50%→30%（寡占との二重計上） |
| **パターンF再定義（戦略混同）** | NFLX#3, #5 | 「専業モデル」「パスワード取締り」は戦略であり優位性ではない → 30% |
| **CAGR掛け算原則** | LLY#6, ATCOA#5 | LLY#6 Global network（CA30%×CAGR50%）に掛け算注記追加。ATCOA#5 M&A CAGR 70%→50% |
| **ブランド類型5/6追加** | MNST, MCO | MNST#4 Multi-brandでブランド類型1（ポートフォリオ運営力）を適用→70%。MCO#6ではブランドが寡占の結果と判定→30% |

### 3.3 全体的傾向

1. **CA平均52.7%はPhase 2実績（50%中央値）と整合的** — Dogma v1.0の「まあ納得（50%）が多い」傾向を再現
2. **T8のCA降格が頻発** — v1.0の「説明力MUST」「戦略≠優位性」が厳格に適用され、3銘柄で合計5件の降格
3. **90%は5件/81主張（6.2%）** — Phase 2実績の6%と完全一致。全てRule 11（業界構造合致）
4. **CAGR > CA の傾向が維持** — 全銘柄平均でCAGR 60.8% > CA 52.7%（Phase 2と同じ傾向）

## 4. 非US銘柄（ATCOA）のデータ品質

| 項目 | v0.9 | v1.0 |
|------|------|------|
| data_source | `analyst_report_extracted` | `ir_filing_extracted` |
| 連結財務 | Q2 2025のみ（レポート記載分） | FY2024 + FY2025 + Q4 2024/2025 |
| セグメント | Q2 2025 OG/OPMのみ | FY2025マージン追加（CT 24.4%, VT 17.2%, IT 20%） |

## 5. フィードバック依頼事項

### アナリストYへの確認事項

1. **各銘柄の確信度は妥当か** — 特にVRSK（最高64%）とATCOA（最低41%）の位置づけ
2. **v1.0変更により改善された評価はあるか** — 特にRule 4「説明力MUST」の適用感
3. **v1.0変更により悪化した評価はあるか** — T8のCA降格（MSFT, MCO, POOL, NFLX）は妥当か
4. **ATCOA（非US）のデータ品質は十分か** — IR資料補強の効果と限界
5. **LLY新規Claim#4（GLP-1供給障壁70%）** — KB3に統合すべきか
6. **POOL#5「アナリスト自己否定」パターン** — dogma v2.0候補として検討すべきか
7. **90%評価の妥当性** — VRSK×2, ORLY×2, MCO×1 の5件は適切か

### 今後の改善提案

1. **KB3拡充**: VRSK, CPRT, MCO をPhase 2検証対象に追加し、KB3の多様性を向上
2. **非US銘柄対応**: ATCOA型のIR資料補強プロセスを標準化
3. **Pattern G対策**: 純粋競合比較の不足（6/12銘柄で検出）に対する入力データ改善
