## 1. 銘柄概要

**Copart, Inc.（CPRT）** は、廃車・中古車のオンラインオークションプラットフォームを運営する米国企業。北米サルベージオークション市場でIAA（RB Global傘下）と合わせて80%の複占市場を形成し、約40-50%のシェアを持つ。1982年創業、1994年ナスダック上場。

- **事業構造**: サルベージオークション（売上75%）、中古車オークション（10%）、海外事業（15%）
- **売り手**: 損保会社が80%、その他金融機関、ディーラー等
- **買い手**: 解体業者が中心、修理業者、中古車ディーラー、輸出業者等
- **収益モデル**: オークションフィー（売り手・買い手双方）+ 搬送・保管・所有権移転サービス
- **直近業績**: FY24/4Q Sales yoy +7.2%, OPM 33.6%, NI Margin 30.1%, EPS $0.34
- **BS**: Total Assets $10.6B, Cash $5.2B, Total Liabilities $962M（極めて健全、実質無借金）
- **CAGR見通し**: EPS CAGR 11%（売上8% + 営業レバ2% + 自社株買い1%、配当ゼロ）

### 主張 #1: ネットワーク効果（40万超のグローバルBidder基盤による流動性・ASP優位）

#### AI評価: おおむね納得（70%）

オークションプラットフォームにおけるネットワーク効果は、構造的な競争優位性属性である（ルール2, ルール6適合）。40万超のBidder基盤は定量的裏付けがあり（ルール4: +20%効果）、IAA対比でのASP成長率格差（CPRT +5.7% vs IAA +1%）は純粋競合との具体的比較として強力な証拠（ルール7: +10-20%効果）。KB3 ORLY#1（AZO対比でのDIFM補充頻度: 70%）と同様に、具体的競合名と比較軸が明確であり70%が妥当。

**納得できる点**: Bidder数の多さとASP格差はネットワーク効果の存在を定量的に裏付ける。海外Bidder増加による国際的なネットワーク拡大は年々優位性を強化する正のフィードバックループ。

**懸念点**: Bidder登録者数は2015年時点（40万超）以降アップデートがなく、11年間の推移が確認できない。ASP格差が構造的（ネットワーク効果由来）か一時的（市場環境による）かの検証が必要。

### 主張 #2: NIMBY資産としてのヤード保有による参入障壁

#### AI評価: おおむね納得（70%）

Yard保有はNIMBY産業における物理的参入障壁であり、構造的な競争優位性属性（ルール2, ルール6適合）。1万エーカー・200施設の定量データ（ルール4）、IAA対比での保有vsリースの差異（ルール7）が明確。ただしCFO面談（2023/6）で「non-salvage比率増加によりyard保有の重要性が変遷し、競争優位性が中長期的に希薄化する可能性」が指摘されており、この構造変化リスクを反映して70%。

**納得できる点**: NIMBY資産による参入障壁は模倣困難性が高い。Yardの新規取得には長期間の行政折衝が必要であり、大都市近辺の立地は「イス取りゲーム」。CPRTは保有（資産計上）、IAAはリース主体という明確な差異。50%の余剰キャパシティ（RB主張、検証困難）も保有側の戦略的余裕。

**懸念点**: アナリスト自身が「non-salvageへのシフトによってyard保有の重要性が変遷し、競争優位性が中長期的に希薄化する可能性」を認識（S6）。中古車は移動可能であり必ずしもYardは必要ない。

**ルール適用マトリクス**:

| R1: 能力vs結果 | Pass | 物理的資産＝能力の源泉 |

| R3: 相対的優位性 | Pass | IAA対比で保有vsリース |
| R4: 説明力 | Pass | 1万エーカー、200施設 |
| R6: 構造的vs補完的 | Structural | サルベージ事業の核心的資産 |
| R7: 純粋競合差別化 | Pass | IAA（リース主体）との差異 |
| R10: ネガティブケース | Warning | non-salvageシフトリスク |

### 主張 #3: 保険会社とのプロセス統合によるスイッチングコスト

#### AI評価: おおむね納得（70%）

スイッチングコストは典型的な競争優位性属性（ルール2）。プロセス統合・人材配置・Express Titlingの具体的メカニズム（ルール1, ルール4）、10大保険会社のうち7社契約/4社独占の定量データ（ルール4: +20%）あり。GEICOのIAA→CPRTスイッチ事例は「スイッチは稀にしか起きない」ことの裏付け（ルール10の間接活用）。KB3 COST#1（退職率9% vs 業界20+%: 70%）と同等の構造。

**納得できる点**: Express Titling（所有権移転代行、run rate 1M台）はCPRT独自のサービスであり、保険会社のコア業務集中を可能にする付加価値。プロセス統合と人材配置の粘着性は高い。

**懸念点**: RB Global面談（2025/10）で「rebateや接待でシェア奪還」を目指す姿勢が示されており、スイッチングコストは絶対的ではない。USAAの喪失事例もある。Erie InsuranceのRB→CPRT移行事例は逆にスイッチが起きうることの証左でもある。

| R4: 説明力 | Pass | 7/10社契約、4社独占、ET 1M台 |

| R7: 純粋競合差別化 | Partial | IAA対比でExpress Titlingは差別化 |
| R10: ネガティブケース | Pass | GEICOスイッチは稀の裏付け |
| R12: 期初vs四半期 | Pass | 期初レポートで主要優位性 |

### 主張 #4: 複占市場構造における構造的ポジション

#### AI評価: まあ納得（50%）

複占市場構造は業界構造そのものであり、CPRT固有の「能力・仕組み」ではない（ルール1抵触リスク）。市場シェア4割超は結果であり能力ではない（パターンA: 結果を原因と取り違えリスク）。KB3 MNST#1（シェアはあくまで結果→50%）と類似。ただし複占構造は参入障壁の高さを示す業界環境であり、Claim#1-3の優位性が複占構造の中で機能している。

**納得できる点**: 複占市場は新規参入が極めて困難（NIMBY + ネットワーク効果 + 規制の三重障壁）。価格競争が緩和されマージンが維持される。

**懸念点**: 複占構造はCPRT固有の能力ではない。IAAも同じ構造的恩恵を受けている。シェアの高さは結果であり、背後の能力はClaim#1-3で評価済み。

**CAGR接続（70%）**: 複占市場 → 新規参入困難 → マージン維持 → 営業レバレッジの基盤。

### 主張 #5: TLF（全損判定率）上昇による構造的な量的成長ドライバー

#### AI評価: まあ納得（50%）

TLF上昇はCPRT固有の競争優位性ではなく業界全体の追い風（マクロドライバー）。ルール1の観点から「外部環境の変化」であり「能力・仕組み」ではない。ただしTLF上昇トレンドの分析は緻密（1981年4%→2024年21.4%→中期30%見込み）であり、CAGR前提の根拠として充実。

**納得できる点**: 40年間の時系列データに裏付けられた構造的トレンド。車齢上昇・電装化・新興国需要という複数のドライバーが相互補完。

**懸念点**: 業界全体に共通する追い風であり差別化要因ではない（ルール3抵触）。自動運転・ADASの進展による事故率低下リスクは長期的な不確実性。

**CAGR接続（90%）**: TLF上昇 → 全損処理増加 → Units成長（4%のうち3%がTLF寄与）。CCC Crash Course Reportで定期確認可能。直接的メカニズム（1-2ステップ）+ 開示データで検証可能 + 業界特性合致。KB3 COST顧客粘着力CAGR（90%）と同等。

## 4. 総合評価

### 競争優位性の構造

```
【核心的優位性（三位一体）】
  #1 ネットワーク効果（70%） ←→ #3 スイッチングコスト（70%）
         ↕                              ↕
  #2 NIMBY Yard（70%）          Express Titling（追加粘着性）
         ↕
  #4 複占市場構造（50%）        --- 基盤条件（能力ではなく環境）

【CAGRドライバー】
  #5 TLF上昇（50%/CAGR90%）   --- 業界追い風（Units成長の主要ドライバー）
  #6 スケールメリット（50%/CAGR70%） --- 営業レバレッジ
  #7 海外横展開（50%/CAGR70%） --- 成長の地理的多様化
```

### 強み

1. **三位一体の相互強化構造**: ネットワーク効果、NIMBY Yard、スイッチングコストが相互に強化し合い、参入障壁を構築。
2. **定量的裏付けの充実**: ASP格差、Yard面積、保険会社契約数、TLF推移など、主要主張に数値データが存在。
3. **純粋競合（IAA/RB）との具体的比較**: ASP成長率格差、土地保有vsリース、Express Titling等で差異が明確。

### 弱み・留意点

1. **non-salvageシフトリスク**: 中古車比率の上昇によりYard保有の重要性が低下する可能性。アナリスト自身が認識。
2. **IAA/RBのrebate戦略**: 価格競争やrebateによるシェア侵食リスク。ただしASP格差が示すようにネットワーク効果の差が防波堤。
3. **Bidder数の情報更新不足**: 2015年以降の更新なし。ネットワーク効果の定量的検証に限界。
4. **スケールメリットの純粋競合差別化不足**: IAA対比でのスケール差異が定量的に示されていない。

### KB3との対比

CPRTの主張構造は、ORLY（平均CA 63%: フラグメント市場×規模・密度）とは異なる「複占市場×ネットワーク効果×NIMBY」構造。ORLYで90%を獲得した「業界構造×企業ポジション合致」の深度には達していないが、3つの70%主張が相互補完する構造は堅牢。最も近い類似はCHD（平均CA 50%）で、能力vs結果の区別や定量裏付けの重要性が共通するが、CPRTは純粋競合比較（IAA）がCHD（明確な純粋競合なし）より充実している分、平均スコアが高い。
