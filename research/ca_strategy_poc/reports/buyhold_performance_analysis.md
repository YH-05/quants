## 2. リスクフリーレートの改善

### 2.1 従来手法の問題点

従来は年率4.5%固定のリスクフリーレートを使用していた。しかし、分析期間（2016-2026）の実際の米国10年債利回りは大きく変動している：

| 期間   | DGS10 水準 | 固定4.5%との乖離 |
| ------ | :--------: | :--------------: |
| 2016年 | 1.5〜2.6%  | **-2.0〜-3.0%**  |
| 2017年 | 2.0〜2.6%  | **-1.9〜-2.5%**  |
| 2018年 | 2.5〜3.2%  | **-1.3〜-2.0%**  |
| 2019年 | 1.5〜2.8%  | **-1.7〜-3.0%**  |
| 2020年 | 0.5〜1.9%  | **-2.6〜-4.0%**  |
| 2021年 | 0.9〜1.7%  | **-2.8〜-3.6%**  |
| 2022年 | 1.5〜4.2%  | **-0.3〜-3.0%**  |
| 2023年 | 3.3〜5.0%  | **-0.5〜+0.5%**  |
| 2024年 | 3.6〜4.7%  | **-0.3〜+0.2%**  |
| 2025年 | 4.2〜4.8%  | **-0.3〜+0.3%**  |

固定4.5%は2023-2025年の水準に近いが、2016-2021年の低金利環境では実態から大きく乖離しており、超過リターンを過小評価していた。

### 2.2 新手法：FRED DGS10 日次データ

| 項目         | 値                                                                    |
| ------------ | --------------------------------------------------------------------- |
| データソース | FRED（Federal Reserve Economic Data）                                 |
| シリーズID   | DGS10                                                                 |
| 定義         | Market Yield on U.S. Treasury Securities at 10-Year Constant Maturity |
| 頻度         | 日次                                                                  |
| 観測数       | 2,557                                                                 |
| 期間         | 2015-12-01 ~ 2026-02-24                                               |
| 最小値       | 0.52%（2020年8月）                                                    |
| 最大値       | 4.98%（2023年10月）                                                   |
| 期間平均     | 2.71%                                                                 |

**日次変換式**: `rf_daily = (1 + DGS10/100)^(1/252) - 1`

各営業日のDGS10値を日次利率に変換し、Sharpe・Sortino・CAPMアルファの計算に使用。非営業日はforward-fillで補完。

### 2.3 指標への影響

| 指標                   | 固定4.5% | DGS10 (2.71%) |  変化  | 要因                            |
| ---------------------- | :------: | :-----------: | :----: | ------------------------------- |
| **Sharpe (30)**        |  0.600   |     0.711     | +0.111 | Rf低下→超過リターン増           |
| **Sharpe (60)**        |  0.603   |     0.700     | +0.097 | 同上                            |
| **Sharpe (90)**        |  0.577   |     0.678     | +0.101 | 同上                            |
| **Sortino (30)**       |  0.538   |     0.647     | +0.109 | 同上                            |
| **Alpha vs MSCI (30)** |  +1.99%  |    +2.37%     | +0.38% | Rf低下→ベータ調整時の期待値変化 |
| **Alpha vs MSCI (60)** |  +1.81%  |    +1.91%     | +0.10% | 同上（β≈1で影響小）             |
| **Alpha vs MSCI (90)** |  +1.24%  |    +1.39%     | +0.15% | 同上                            |

**解説**: Sharpeは `(R_p - R_f) / σ` で計算されるため、R_fが4.5%→2.71%に下がると分子の超過リターンが増加し、Sharpeが改善する。CAPMアルファは `α = R_p - R_f - β(R_b - R_f)` であり、β<1のポートフォリオ（30-stock: β=0.787）ではR_f低下のα改善効果が大きい。


**解説**: β<1のポートフォリオほどR_f低下のAlpha改善効果が大きい。30-stock（β=0.787）は1-β=0.213が大きいため、R_fが1.79%低下するとαが+0.38%改善する。60-stock（β≈0.944）では1-β≈0.056で改善幅は+0.10%に留まる。

## 9. 方法論詳細

### 9.1 Buy-and-Hold ドリフトウェイト

日次のウェイト更新:

```
w_i(t+1) = w_i(t) × (1 + r_i(t)) / Σ_j[w_j(t) × (1 + r_j(t))]
```

初期ウェイトはCA Strategyのスコアベースポートフォリオ構築結果を使用し、期間中リバランスを行わない。銘柄のコーポレートアクション発生時のみ、当該銘柄ウェイトを0にして残存銘柄に比例配分。

### 9.2 リスクフリーレート処理

- **データ**: FRED DGS10（米国債10年物利回り、日次、年率%単位）
- **日次変換**: `rf_daily(t) = (1 + DGS10(t)/100)^(1/252) - 1`
- **欠損補完**: 非営業日はforward-fill
- **年率化**: ポートフォリオ分析日と同一日のDGS10値を使用（日ごとに異なるRf）
- **幾何平均**: 期間の年率Rfは `(Π(1+rf_daily))^(252/N) - 1 = 2.71%`

### 9.3 Sharpe Ratio 計算

```
excess_daily(t) = r_p(t) - rf_daily(t)   # 日次DGS10使用
Sharpe = mean(excess_daily) / std(excess_daily) × √252
```

### 9.4 CAPM Alpha 計算

```
β = Cov(r_p, r_b) / Var(r_b)            # 日次リターン共分散
R_f = 期間幾何平均Rf = 2.71%
α = R_p(ann) - R_f - β × (R_b(ann) - R_f)
```
