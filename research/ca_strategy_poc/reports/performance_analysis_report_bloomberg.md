# CA Strategy パフォーマンス分析レポート（Bloomberg データ版）

**作成日**: 2026-02-27
**分析期間**: 2016年1月〜2026年2月（約10.5年間、2,648営業日）
**データソース**: Bloomberg Terminal（PX_LAST: 分割調整済み価格、配当未含）
**リスクフリーレート**: FRED DGS10（米10年国債利回り、年率換算 2.71%）

---

## エグゼクティブ・サマリー

Bloomberg データを用いた CA Strategy の10年間 Buy-and-Hold バックテスト結果を報告する。本レポートでは、CA Strategy ポートフォリオ（30/60/90銘柄）、投資ユニバース（MCap加重/等加重）、MSCI Kokusai（時価総額加重/等加重）の6つの戦略を横断的に比較する。

### データソース比較

| 項目 | yfinance版 | Bloomberg版 |
|------|-----------|-------------|
| ユニバースカバレッジ | 278/395 (70.4%) | 370/395 (93.7%) |
| MSCI Kokusai ベンチマーク | TOK ETF（プロキシ） | MXKO INDEX（直接） |
| 時価総額データ | 初期時点のみ（静的） | 月次ヒストリカル |
| 価格データ種別 | 配当調整済み（Total Return） | 分割調整のみ（Price Return） |

### 主要な発見

1. **投資ユニバース（MCap加重）は MSCI Kokusai をアウトパフォーム**: 年率 +0.90%、Alpha +1.84%、Sharpe 0.632 vs 0.535。ユニバースの銘柄選定基盤自体に付加価値がある。
2. **CA Strategy ポートフォリオは MSCI Kokusai をアンダーパフォーム**: 年率 -1.32%〜-2.09%。ただし配当未含のため、配当込みでは縮小する可能性がある。
3. **CA Strategy は等加重ベンチマーク対比ではアウトパフォーム**: MSCI Kokusai EW 対比 +0.82%〜+1.59%、ユニバース EW 対比 +0.69%〜+0.76%（60/90銘柄）。スコアウェイティングの有効性を示唆。
4. **ディフェンシブ特性は Bloomberg データでも確認**: Beta 0.83〜0.89、下落局面（2018, 2022年）での優位性は健在。
5. **2023-2025年の AI/テック相場で全戦略が MCap加重に劣後**: 時価総額集中効果が支配的。

### 1.1 Bloomberg PX_LAST vs yfinance Adjusted Close

**注目点**:
- **Univ MCap のみがポジティブな Capture Spread（+2.4%）**を示し、市場上昇時により多く獲得し下落時により抑制。これがアウトパフォーマンスの構造的要因。
- CA Strategy ポートフォリオは Capture Spread ≈ 0 で、ディフェンシブ特性はあるが上昇キャッチアップが不十分。
- yfinance版では AVGO の寄与により CA Strategy もポジティブな Capture Spread を示していたが、Bloomberg版（AVGO欠如）では消失。

| vs MSCI Kokusai（MCap） | **アンダーパフォーム** | -1.3%〜-2.1%/年、ネガティブ IR |
| vs MSCI Kokusai EW | **アウトパフォーム** | +0.8%〜+1.6%/年、ポジティブ Alpha |
| vs ユニバース MCap | **アンダーパフォーム** | -2.2%〜-3.0%/年、ネガティブ Capture Spread |
| vs ユニバース EW | **アウトパフォーム**（60/90銘柄） | +0.7%〜+0.8%/年、ポジティブ IR |
| 下落耐性 | **有効** | 2018年 +3.4%〜+7.4%、2022年 +0.5%〜+5.0% |
| テック相場追随 | **構造的に劣後** | 2023-2025年 -5%〜-12%/年 |

**結論**: CA Strategy は等加重戦略に対してはスコアウェイティングによる銘柄選択効果を発揮するが、時価総額加重戦略（MSCI Kokusai, MCap ユニバース）に対しては、大型テック株の Buy-and-Hold drift 効果に追随できず劣後する。ディフェンシブ特性は堅牢だが、2023-2025年のようなテーマ集中型の上昇相場では構造的なアンダーパフォーマンスが生じる。

### 8.3 投資ユニバースの付加価値

投資ユニバース（MCap加重）は MSCI Kokusai を年率 +0.90%（Alpha +1.84%）アウトパフォームしており、銘柄選定基盤自体に価値がある。CA Strategy の課題は、ユニバースから「良い銘柄を選ぶ」段階ではなく、「ウェイトの付け方」（スコアウェイティング vs 時価総額加重）にある。

### 8.4 今後の改善方針

1. **Total Return Index の取得**: Bloomberg `TOT_RETURN_INDEX_GROSS_DVDS` フィールドで Total Return データを取得し、yfinance版と条件を統一
2. **欠如銘柄のマッピング改善**: AVGO, MHFI, DD 等の後継ティッカーを Bloomberg FIGI / CUSIP で正確にマッピング
3. **リバランス戦略の評価**: Buy-and-Hold drift の代わりに、四半期/年次リバランスのバックテストも実施
4. **ウェイティング手法の再検討**: スコアウェイティングに時価総額ティルトを組み合わせたハイブリッド方式の検討
