## 1. エグゼクティブサマリー

本レポートは、CA（Competitive Advantage）Strategyに基づいて構築された90銘柄ポートフォリオの詳細分析を提供する。CA Strategyは、MSCI Kokusai構成銘柄313銘柄の決算トランスクリプトから、AI（Claude Sonnet 4）を活用して競争優位性に関する主張（claim）を抽出・スコアリングし、セクター中立制約の下で上位銘柄を選定する体系的な投資手法である。

90銘柄ポートフォリオは、30銘柄（集中型）・60銘柄（バランス型）に続く**広域分散型ポートフォリオ**として、313銘柄ユニバースの28.8%をカバーする。ユニバース全体のスコア平均0.5289に対し、ポートフォリオのスコア平均は0.6380と、+0.1091（+20.6%）のスコアプレミアムを確保している。30銘柄版のスコアプレミアム+28.3%、60銘柄版の+24.4%と比較すると、銘柄数の拡大に伴うスコア希薄化が進行しているが、依然としてユニバース平均を20%超上回る水準を維持している。

セクター配分はベンチマーク（MSCI Kokusai）に完全一致させるセクター中立化を採用しており、超過リターンは純粋な「銘柄選択」に帰属する。HHI 0.0112、有効銘柄数89.5であり、事実上の均等加重に近い分散が実現されている。最大ウェイトはDD（Materials）の1.38%に過ぎず、個別銘柄リスクのインパクトが構造的に抑制されている。

ポートフォリオ最高スコアはPPL（0.7933、Utilities）であり、英国配電事業の規制独占という構造的に極めて強い競争優位性が評価された。最低スコアはSOL（0.5514、Energy）で、GTL技術のユニーク性は認められたものの、低油価環境での収益性低下と大型建設プロジェクトリスクがスコアを制約している。

claim分析では、全702件のclaimのうちcompetitive_advantage（471件、67.1%）が最多を占め、cagr_connection（140件、19.9%）、factual_claim（91件、13.0%）が続く。エビデンスカバレッジは80.9%（568/702）と高水準を維持している。確信度分布では、90%確信度が70件、70%確信度が315件と、中〜高確信度のclaimが主要な構成要素となっている。

## 2. ポートフォリオ構築手法

### 2.1 CA Strategyの概要

CA Strategyは、「企業の長期的な超過リターンは持続可能な競争優位性（Competitive Advantage）に起因する」という投資哲学に基づく。Hamilton Helmerの「7 Powers」フレームワーク（Scale Economies, Network Economies, Counter-Positioning, Switching Costs, Branding, Cornered Resource, Process Power）を基盤としたナレッジベース（KB1-T / KB2-T / KB3-T）と、11のルールからなる独自の評価教義（dogma.md）がスコアリングの一貫性と厳格性を担保する。

### 2.2 5フェーズパイプライン

**Phase 0: 前処理** -- S&P Capital IQの月次決算トランスクリプトJSONを銘柄単位に整理する。Point-in-Time（PoiT）制約により、2015-12-31以前に公開されたトランスクリプトのみを使用し、ルックアヘッドバイアスを完全に防止する。

**Phase 1: 主張抽出** -- Claude Sonnet 4がトランスクリプトを読み込み、KB1-T（抽出ルール）とKB3-T（few-shot例）を参照しながら、銘柄あたり5-15件の競争優位性に関する主張（claim）を抽出する。各claimはclaim_type（competitive_advantage / cagr_connection / factual_claim）に分類され、エビデンスとなるトランスクリプト原文が付与される。

**Phase 2: スコアリング** -- 抽出されたclaimに対し、KB1-T / KB2-T（パターン集: 高評価パターンI-V と却下パターンA-G）/ KB3-T / dogma.mdを参照しながら、10-90%の確信度スコアを付与する。

**Phase 3: 集約・中立化** -- ScoreAggregatorが銘柄単位でスコアを集約し、SectorNeutralizerがセクター内Z-scoreを計算してセクター間のスコア水準差異を中立化する。

**Phase 4: ポートフォリオ構築** -- PortfolioBuilderがベンチマークのセクターウェイトを参照し、各セクター内で上位N銘柄を選定する。セクター内のウェイト配分はスコア比例で行い、セクター合計ウェイトはベンチマークに完全一致させる。

**Phase 5: 出力生成** -- OutputGeneratorがポートフォリオのJSON / CSV / Markdown / 銘柄別rationaleファイルを生成する。

### 2.3 セクター中立化の仕組み

セクター中立化はCA Strategyの重要な設計決定である。各銘柄のセクター内Z-score（Z-score(i) = (Score(i) - Mean(sector)) / Std(sector)）に基づいてセクター内ランキングを決定し、上位銘柄を選定する。最終的なウェイトは、セクター合計ウェイトをベンチマークに一致させた上で、セクター内ではスコア比例配分とする。この結果、ポートフォリオのアクティブリスクは純粋に銘柄選択に帰属し、セクターアロケーション効果はゼロとなる。

## 5. 銘柄選定の傾向と特徴

### 5.1 高スコア銘柄の共通特性

ポートフォリオ上位10銘柄（スコア0.69以上）の競争優位性パターンを分析すると、以下の共通特性が浮かび上がる。

**規制的・構造的参入障壁の保有**: PPL（0.7933）の配電規制独占、MHFI（0.7400）のNRSRO認定、CCEP（0.7000）のCoca-Cola独占ボトラー権など、規制や契約による参入障壁を持つ銘柄が高スコアを獲得する傾向が顕著である。これらの優位性は競合による模倣が制度的に不可能であり、dogmaのRule 1（「結果ではなく原因を示せ」）とRule 3（「業界共通でないことを示せ」）を容易にクリアする。

**ネットワーク効果の存在**: AMS（0.7333）のGDS両面市場（グローバルシェア42.2%）、AAPL（0.7286）のiOSエコシステム（10億台デバイス）、SAP（0.7042）のBusiness Network（1.9百万社、年間$730B取引）など、プラットフォーム型ビジネスにおけるネットワーク効果がスコアを押し上げている。

**定量的エビデンスの豊富さ**: 高スコア銘柄は例外なく明確な定量データをトランスクリプト内で提示している。DIS「11フランチャイズが$1B超」、AVGO CEOの「nobody comes even close at this point」（FBAR技術独占）、BA CEOの「there is no competition for the airplane」（777X）、CLの「ブラジル72.1%シェア」などが典型であり、KB2-TのPattern I（定量的差別化）による+10%確信度調整の恩恵を受けている。

これら3条件のうち2つ以上を満たしている点が、上位10銘柄に共通する特徴である。

### 5.2 60→90で追加された29銘柄のスコア傾向

追加29銘柄の平均スコア0.596は、60銘柄版の最低スコア近辺に位置する。これらの銘柄を競争優位性タイプ別に分類すると、**規模/シェア型**（PG、JPM、005930等9銘柄）が最多であり、「シェアの大きさ自体はCA Strategyで高スコアを得にくい」というパターンが確認される。dogmaは「なぜそのシェアが持続するのか」の構造的説明を要求するため、シェア事実のみではスコアが伸びない。

次いで**ニッチ支配型**（ISRG、ANSS、VRSN等8銘柄）が多く、明確なmoatを持つがPoiT時点のトランスクリプトで十分な成長証拠が抽出されなかったケースが目立つ。**ブランド/フランチャイズ型**（MAR、HSY、BATS等7銘柄）はブランド優位性の定量化が難しく確信度が抑えめになる傾向がある。

### 5.4 「メガキャップの復帰」現象

JPM、PG、005930（Samsung）、SBUX、Visaといったメガキャップ銘柄が90銘柄版で初めてポートフォリオに加わった。大企業のトランスクリプトは事業セグメントが多岐にわたり、競争優位性の主張が分散する傾向がある。また、「誰もが知っている」優位性は経営陣がトランスクリプトで積極的に言語化しないことが多い。しかし、これらがユニバース上位29%に位置すること自体は、CA Strategyの品質フィルタが機能している証拠である。

## 7. リスク特性とリミテーション

### 7.1 1銘柄セクターのリスク

Materials（DD 1銘柄は30銘柄版、90銘柄版では2銘柄に拡大）を除き、Utilities（PPL 1銘柄）とTelecommunication Services（SCMN 1銘柄）は依然として1銘柄セクターであり、当該銘柄固有のイベントがセクターパフォーマンスに直結する。PPLの英国RIIO-ED1レビューリスク、SCMNのスイス通信規制変更リスクが個別に識別される。

### 7.2 低スコア銘柄の含有

90銘柄ポートフォリオ固有のリスクとして、スコア0.55-0.58の銘柄（約15銘柄）の存在がある。SOL（0.551）、RADL3（0.554）、EOG（0.557）、SPG（0.560）、BATS（0.562）等はユニバース平均をわずかに上回る程度であり、CA Strategyの品質フィルタとしての有効性は限界的である。特にエネルギーセクター5銘柄中2銘柄（EOG、SOL）がこの帯域に位置し、セクター中立化制約による強制的な保有である。

### 7.3 Point-in-Time制約

スコアリングに使用するトランスクリプトは2015年9月30日までの情報に基づいており、2015年Q4の情報は含まれない。特に2015年末の原油価格$37/bbl到達、Fed利上げ開始等の情報がスコアに反映されていない。また、Claude Sonnet 4によるclaim抽出はトランスクリプトに明示的に記載された情報に依存し、経営陣が意図的に開示しない競争優位性は抽出されない。

### 7.4 ベンチマークへの接近

90銘柄がユニバース313銘柄の29%をカバーすることで、ポートフォリオはベンチマークに構造的に接近する。Active Shareは30/60銘柄版より確実に低く、クローゼット・インデクシングの懸念が生じうる。ただし、Tracking Error 7.34%は一般的なアクティブ運用ファンドの範囲内（4-8%）であり、アクティブ運用としての性格は維持されている。

### 7.5 LLMバイアスの可能性

英語トランスクリプトへの親和性、特定産業・ビジネスモデルへの理解度の偏りがスコアに影響する可能性がある。ブラジル（RADL3）、韓国（005930）、南アフリカ（SOL）等の非英語圏銘柄のスコアが体系的に低い傾向があるかは、追加的な検証が必要である。

### 7.6 時点固有リスク

本ポートフォリオは2015-12-31時点のスナップショットであり、以下の時点固有リスクが存在する。

- **原油価格暴落**: 2015年は原油価格が$37/bblまで下落した年であり、Energyセクター（ENB、SLB、FTI、EOG、SOL）のスコアは低原油価格環境を反映している。SOLのHEPS -17%（油価-33%環境）がその影響を端的に示す。
- **欧州銘柄の為替リスク**: CCEP、AMS、BAYN、CPI、SCMN、ML、NOVN等の欧州銘柄はEUR/CHF/GBPの為替変動リスクを内包するが、CA Strategyのスコアリングでは為替リスクは明示的に考慮されていない。
- **Fed利上げ開始**: 2015年12月のFRB利上げ開始は金融セクター（JPM、WFC、CBA等）の収益環境に影響するが、PoiT制約によりスコアに反映されていない。
- **中国経済減速**: 2015年夏の中国株急落と経済成長鈍化は、005930（Samsung）のモバイル事業やHSYの中国展開に影響するリスク要因である。

## 8. 結論

CA Strategy 90銘柄ポートフォリオは、313銘柄ユニバースから上位29%の銘柄を体系的に選定した広域分散型ポートフォリオである。ポートフォリオのスコア平均0.6380はユニバース平均0.5289を20.6%上回り、HHI 0.0112（有効銘柄数89.5）は事実上の均等加重に近い分散を実現している。

全10セクター90銘柄のclaimを精査すると、702件の主張のうち67.1%が競争優位性の直接的主張であり、エビデンスカバレッジ80.9%は分析の信頼性を裏付けている。PPLの規制独占（90%確信度）からSOLのGTL技術（50-70%確信度）まで、多様な競争優位性パターンがポートフォリオ全体に分散しており、特定の優位性タイプへの偏りは限定的である。

30/60銘柄版との比較は、分散化の逓減効果（HHI改善: 49%→32%）とスコア品質のトレードオフ（スコアプレミアム: 28.3%→24.4%→20.6%）を明確に示している。90銘柄版はTracking Error 7.34%と最も安定的なリスク特性を持ち、トラッキングエラー制約の厳しい機関投資家向けのユースケースに適合する。一方、Information Ratio 0.247は60銘柄版（0.342）を下回り、アクティブリスクの効率性では60銘柄版が優位である。

3つのポートフォリオサイズは、異なる運用ニーズに対応する階層構造として理解すべきである。30銘柄版はハイコンビクション運用に適し、スコアプレミアム+28.3%の高品質集中投資を提供する。60銘柄版はInformation Ratio 0.342でリスク調整後リターンの最大化を実現し、コア・アクティブ戦略として最も効率的である。90銘柄版は広域カバレッジとリスク分散を重視し、Tracking Error 7.34%の安定性と個別銘柄リスクの構造的抑制（最大ウェイト1.38%）を提供する。

CA Strategyの「競争優位性に基づく銘柄選定」という信号が、90銘柄規模でも年率+2.15%のアクティブリターンを生み出していることは、銘柄数の拡大に伴うスコア希薄化を許容してもなお、戦略の有効性が維持されていることのエビデンスである。702件のclaimに裏付けられた体系的な競争優位性評価は、PPLの規制独占からDISのIPポートフォリオ、AMSのGDSネットワーク効果、ISRGのda Vinciプラットフォーム、Visaの決済ネットワークまで、Hamilton Helmerの7 Powersフレームワークに基づく多様な優位性パターンを網羅的に捕捉している。この多様性こそが、90銘柄ポートフォリオの構造的強みであり、特定の優位性タイプの陳腐化リスクに対する天然のヘッジとして機能する。
